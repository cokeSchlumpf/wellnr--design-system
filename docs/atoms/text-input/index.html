Introduction Foundations Components Introduction Foundations Components Atoms Button Code Container Grid Heading Markdown Pane Paragraph Pre Tabs Text Input # Text Input The Text Input component allows user to type in text. `w-text-input` supports common attributes like the HTML `input`-Tag like `value` or `placeholder`. With `clearable` an additional button to clear the value is added. Use the `invalid` attribute to indicate that the field content is invalid.
